    Roadmap

    pg_graphql aims to implement all of the GraphQL core specification and as much of the relay server specification as is practical.

    • Fragments
    • Variables
    • Named Operations
    • Introspection Schema

    At time of writing, the introspection schema is a partial implementation. It is complete enough to render the GraphiQL explorer & to dump the schema via the Apollo CLI.
